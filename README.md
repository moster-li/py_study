# py_test
### Python学习计划（四周）

#### 目标：
- 掌握Python的核心概念，能够独立构建项目。
- 通过实践项目巩固所学知识，培养独立解决问题的能力。

---

### **第一周：Python基础语法与数据结构**
**目标**：掌握Python的基本语法和常用数据结构，为后续学习打下基础。

1. **Day 1-2: Python基础语法**
   - 变量与数据类型（整数、浮点数、字符串、布尔值）
   - 输入与输出（`input()`、`print()`）
   - 基本运算符（算术、比较、逻辑）
   - 注释与代码风格（PEP 8）

2. **Day 3-4: 控制流程**
   - 条件语句（`if`、`elif`、`else`）
   - 循环（`for`、`while`）
   - `break`、`continue`、`pass`

3. **Day 5-6: 数据结构**
   - 列表（`list`）：创建、访问、修改、常用方法（`append()`、`remove()`、`sort()`等）
   - 元组（`tuple`）：不可变性与使用场景
   - 字典（`dict`）：键值对、常用方法（`keys()`、`values()`、`items()`）
   - 集合（`set`）：去重、集合运算

4. **Day 7: 综合练习**
   - 编写一个简单的计算器程序，支持加减乘除。
   - 编写一个程序，统计一段文本中每个单词出现的次数。

---

### **第二周：函数与模块**
**目标**：掌握函数的定义与使用，理解模块化编程的思想。

1. **Day 1-2: 函数**
   - 函数的定义与调用
   - 参数与返回值
   - 局部变量与全局变量
   - 匿名函数（`lambda`）

2. **Day 3-4: 模块与包**
   - 导入模块（`import`）
   - 常用内置模块（`math`、`random`、`datetime`）
   - 创建与使用自定义模块
   - 包的概念与使用

3. **Day 5-6: 错误处理**
   - 异常处理（`try`、`except`、`finally`）
   - 自定义异常

4. **Day 7: 综合练习**
   - 编写一个程序，生成随机密码（使用`random`模块）。
   - 编写一个程序，处理用户输入的数字，确保输入有效（使用异常处理）。

---

### **第三周：面向对象编程（OOP）与文件操作**
**目标**：理解面向对象编程的基本概念，掌握文件读写操作。

1. **Day 1-2: 面向对象编程基础**
   - 类与对象
   - 属性与方法
   - 构造函数（`__init__`）
   - 继承与多态

2. **Day 3-4: 文件操作**
   - 文件的读写（`open()`、`read()`、`write()`）
   - 文件路径处理（`os`模块）
   - `with`语句与上下文管理器

3. **Day 5-6: 高级OOP概念**
   - 类方法与静态方法
   - 私有属性与封装
   - `__str__`与`__repr__`

4. **Day 7: 综合练习**
   - 编写一个简单的学生管理系统，支持添加、删除、查询学生信息（使用类与文件操作）。
   - 编写一个程序，读取一个文本文件并统计其中每个单词的出现次数。

---

### **第四周：常用库与项目实践**
**目标**：掌握常用的Python库，开始构建小型项目。

1. **Day 1-2: 常用库**
   - `os`与`sys`模块：系统操作
   - `json`模块：JSON数据的处理
   - `requests`库：HTTP请求

2. **Day 3-4: 项目实践**
   - 选择一个简单的项目（如天气查询工具），使用`requests`库获取API数据并展示。

3. **Day 5-6: 项目实践**
   - 编写一个简单的待办事项管理器，支持添加、删除、查看任务（使用文件存储数据）。

4. **Day 7: 复习与总结**
   - 复习前几周的内容，查漏补缺。
   - 准备进入项目阶段。

---

### **推荐项目**

#### 1. **项目1：简易计算器**
- **描述**：编写一个命令行计算器，支持加减乘除运算。
- **关键概念**：基础语法、控制流程、函数。
- **难度**：入门

#### 2. **项目2：单词计数器**
- **描述**：编写一个程序，读取文本文件并统计每个单词的出现次数。
- **关键概念**：文件操作、字典、字符串处理。
- **难度**：入门

#### 3. **项目3：学生管理系统**
- **描述**：编写一个简单的学生管理系统，支持添加、删除、查询学生信息，并将数据保存到文件中。
- **关键概念**：面向对象编程、文件操作、异常处理。
- **难度**：中级

#### 4. **项目4：天气查询工具**
- **描述**：使用`requests`库调用天气API，获取并展示当前城市的天气信息。
- **关键概念**：API调用、JSON数据处理、模块化编程。
- **难度**：中级

#### 5. **项目5：待办事项管理器**
- **描述**：编写一个命令行待办事项管理器，支持添加、删除、查看任务，并将任务保存到文件中。
- **关键概念**：文件操作、面向对象编程、异常处理。
- **难度**：进阶

---

### **总结**
- 通过四周的学习，你将掌握Python的核心概念，并能够独立构建小型项目。
- 项目难度逐步递增，帮助你巩固所学知识并培养独立解决问题的能力。
- 完成这些项目后，你可以继续探索更复杂的Python库和框架（如`Flask`、`Django`、`Pandas`等），进一步提升编程能力。
